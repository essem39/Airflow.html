Политика конфиденциальности

Как мы собираем и защищаем ваши данные

Дата последнего обновления: 31 декабря 2025 г.

Настоящая Политика конфиденциальности описывает, как Airflow (поисковый сервис авиабилетов на домене search.airflowtravel.net) собирает, использует и защищает ваши данные при использовании нашего сайта.

Важно: Мы не продаём авиабилеты напрямую. Мы агрегатор поиска, работающий через систему Aviasales/Travelpayouts. При переходе на сайты партнёров (авиакомпаний, агентств) их политики конфиденциальности применяются отдельно.

1. Какую информацию мы собираем

При использовании нашего сайта мы собираем следующие типы данных:

1.1. Поисковые запросы (обезличенные):

  • Город отправления и назначения
  • Даты вылета и возвращения
  • Количество пассажиров и класс обслуживания

Эти данные используются только для поиска билетов и не идентифицируют вас лично.

1.2. Технические данные:

  • IP-адрес (для определения региона и валюты)
  • Тип браузера и операционной системы
  • Разрешение экрана
  • Источник перехода (откуда вы пришли на сайт)

1.3. Данные о поведении на сайте:

  • Просмотренные страницы
  • Время на сайте
  • Кликнутые предложения

2. Как мы используем данные

Собранная информация используется для:

  • Поиска авиабилетов через систему Aviasales
  • Отображения актуальных данных (курсы валют, погода) через бесплатные API
  • Улучшения сервиса - анализ популярных направлений
  • Персонализации - показ результатов в вашей валюте
  • Статистики - общие данные о посещаемости
Мы НЕ собираем:
  • Данные банковских карт
  • Паспортные данные
  • Персональные контакты (email, телефон)
Все платежи происходят на сайтах наших партнёров (авиакомпаний, агентств).

3. Файлы Cookie и трекинг

Мы используем Cookie для следующих целей:

3.1. Необходимые Cookie:

  • Партнёрские метки - для корректного начисления комиссии от Aviasales/Travelpayouts
  • Сохранение настроек - язык, валюта

3.2. Аналитические Cookie:

  • Google Analytics - общая статистика посещений
  • Yandex Metrika - анализ поведения на сайте

3.3. Партнёрские пиксели:

  • Travelpayouts tracking - отслеживание конверсий
  • Aviasales markers - для корректной работы поиска

Вы можете отключить Cookie в настройках браузера, но это может повлиять на работу сайта.

4. Сторонние сервисы и API

Для предоставления качественного сервиса мы используем следующие сторонние API:

4.1. Aviasales (Travelpayouts):

  • Основная система поиска авиабилетов
  • Сравнение цен среди 728 авиакомпаний
  • Их политика конфиденциальности

4.2. ExchangeRate-API:

  • Бесплатное API для курсов валют
  • Обновление каждые 5 минут
  • Не передаёт личные данные

4.3. Open-Meteo:

  • Бесплатное API для погоды
  • Температура в популярных городах
  • Не передаёт личные данные

5. Передача данных третьим лицам

Мы передаём данные только в следующих случаях:

5.1. Партнёрам по поиску:

  • При клике на результат поиска вы переходите на сайт Aviasales или напрямую к авиакомпании
  • Передаются: поисковый запрос, партнёрская метка
  • Далее действуют их политики конфиденциальности

5.2. Требования законодательства:

  • При официальном запросе государственных органов

Мы НЕ продаём ваши данные рекламодателям или третьим лицам!

6. Защита данных

Меры безопасности:

  • SSL-шифрование - все данные передаются по защищённому соединению
  • Обезличивание - мы не храним личную информацию
  • Регулярные обновления - безопасность сервера

7. Ваши права

В соответствии с законодательством, вы имеете право:

  • Знать какие данные мы собираем
  • Запросить удаление ваших данных
  • Отозвать согласие на обработку
  • Получить копию ваших данных

Для реализации ваших прав свяжитесь с нами:

info@airflowtravel.net

8. Дети и конфиденциальность

Наш сервис не предназначен для детей младше 18 лет. Мы сознательно не собираем информацию о детях.

9. Изменения в политике

Мы можем обновлять эту Политику конфиденциальности. Существенные изменения будут опубликованы на этой странице с обновлённой датой.

Продолжая использовать сайт после изменений, вы соглашаетесь с обновлённой Политикой.
